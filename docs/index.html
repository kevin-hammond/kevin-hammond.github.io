Privacy Policy for Club Automation eSigner

Last updated: [Current Date]

Data Collection

Club Automation eSigner does not collect, store, or transmit any user data. All signature processing and PDF generation occurs locally on the user's device.

Data Usage

The extension only processes data locally to:

  • Capture electronic signatures on receipt pages
  • Generate PDF documents with signed receipts
  • Save signed receipts to the user's local device

Data Storage

No data is stored by the extension. All generated PDFs are saved directly to the user's local downloads folder.

Third-Party Services

This extension does not communicate with any third-party services or servers.
